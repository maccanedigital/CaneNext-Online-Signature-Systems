# CaneNext Online Signature System

## วิธีอัปโหลดขึ้น GitHub Pages

ห้ามอัปโหลดไฟล์ ZIP เป็นไฟล์เดียวใน Repository

1. แตกไฟล์ ZIP นี้ก่อน
2. อัปโหลดไฟล์ต่อไปนี้ไว้ที่หน้าแรกของ Repository:
   - index.html
   - app.js
   - config.js
   - style.css
   - .nojekyll
3. ไปที่ Settings > Pages
4. เลือก Deploy from a branch
5. เลือก Branch: main และ Folder: /(root)
6. กด Save
